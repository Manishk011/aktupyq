# AKTU PYQ Frontend

This is a React + Vite + Tailwind CSS project for the AKTU PYQ website.

## setup

Since Node.js was not detected in the environment during creation, the files were generated manually.

To run this project:

1.  **Install Node.js:** Download and install from [nodejs.org](https://nodejs.org/).
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

## Structure

-   `src/components`: UI components
-   `src/pages`: Page views
-   `src/assets`: Static assets
